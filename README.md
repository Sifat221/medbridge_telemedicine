# 🏥 MedBridge Telemedicine App

MedBridge is a Flutter-based telemedicine mobile application designed to connect patients, doctors, and rural health workers in one unified platform. The app enables easy access to healthcare services through digital consultations, appointment booking, and medical record management.

---

## 🚀 Features

### 🔐 Authentication & Security
- User Signup & Login
- OTP Verification
- Forgot & Reset Password
- Secure Token-based Authentication

### 👤 User Roles
- Patient
- Doctor
- Rural Health Worker

---

## 🧑‍⚕️ Patient Features
- Browse and search doctors by specialty
- View doctor profiles and ratings
- Book appointments with available time slots
- Join video consultations
- Real-time chat with doctors
- View prescriptions and download reports
- Access medical history & health records

---

## 👨‍⚕️ Doctor Features
- View incoming patient cases
- Accept or reject case requests
- Add diagnosis and treatment plans
- Write digital prescriptions
- Manage appointment schedules

---

## 🏥 Rural Health Worker Features
- Register and manage patients
- Upload symptoms and vital data
- Upload medical reports (images/PDF)
- Track case status and updates

---

## 💬 Communication
